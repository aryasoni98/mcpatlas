# Roadmap

Aligned with [ROADMAP.md](https://github.com/cncf-mcp/server/blob/main/ROADMAP.md) in the repo.

## Phase 1: Foundation (complete)

- Core MCP server (STDIO, SSE, Streamable HTTP)
- 14 tools, resources, prompts
- Docker, documentation, CI
- Homebrew formula template

## Phase 2: Intelligence (in progress)

- Storage traits (Graph, Vector, Cache)
- SurrealDB, Qdrant, hybrid search
- Artifact Hub, LLM summaries
- CNCF Sandbox application

## Phase 3: Extensibility

- WASM plugin host (Extism)
- Dynamic plugin tools
- Kubernetes operator
- Contributor automation

## Phase 4: Scale & maturity

- Redis shared state
- RBAC, optional auth
- SLSA Level 3, cosign
- Persistent Tantivy index

See the [full roadmap](https://github.com/cncf-mcp/server/blob/main/ROADMAP.md) for success metrics and details.
