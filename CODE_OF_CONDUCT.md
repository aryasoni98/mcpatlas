# Code of Conduct

This project follows the [CNCF Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md).

## Summary

- Be respectful, inclusive, and collaborative.
- Harassment, trolling, and discriminatory behavior are not tolerated.
- Report violations to the project maintainers or to the CNCF at conduct@cncf.io.

By participating, you agree to uphold this code of conduct.
